# Put model files into path

without version isolation
~~~
/.minecraft/EF_Skin/<skin>/<model.json/info.json.....>
~~~

version isolation
~~~
/.minecraft/version/<version_name>/EF_Skin/<skin>/<model.json/info.json.....>
~~~
